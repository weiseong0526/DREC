# 醫生Chang 專業血糖管理專家 - 作品集網站

這是一個為醫生Chang設計的現代化、專業的作品集網站，專注於血糖管理服務。

## 功能特色

- ✨ **現代化設計** - 響應式設計，適配所有設備
- 🎯 **專業展示** - 突出醫生的專業背景與服務項目
- 💬 **真實見證** - 展示患者成功案例
- ❓ **常見問題** - 互動式FAQ區塊
- 📞 **聯絡表單** - 方便患者報名與諮詢
- 🚀 **流暢動畫** - 優雅的頁面過渡效果

## 檔案結構

```
portfolio/
├── index.html      # 主頁面
├── styles.css      # 樣式表
├── script.js       # JavaScript功能
└── README.md       # 說明文件
```

## 使用方式

1. **本地預覽**
   - 直接在瀏覽器打開 `index.html` 檔案
   - 或使用本地伺服器（如 XAMPP）運行

2. **部署到網站**
   - 將所有檔案上傳到您的網站伺服器
   - 確保所有檔案在同一目錄下

## 自訂內容

### 修改顏色主題
在 `styles.css` 的 `:root` 區塊中修改顏色變數：
```css
:root {
    --primary-color: #2c5f8d;    /* 主色調 */
    --secondary-color: #d4a574;   /* 次要色調 */
    --accent-color: #e74c3c;      /* 強調色 */
}
```

### 更新聯絡資訊
在 `index.html` 的聯絡區塊中修改地址、電話和電子郵件。

### 表單提交
目前表單使用前端處理。要連接後端：
1. 在 `script.js` 中找到表單提交處理
2. 取消註解後端API呼叫部分
3. 設定您的API端點
